# Cudiary Web Application

This is the web frontend for the Cudiary application.

## Structure

- `public/` - Static files and HTML entry point
- `src/` - Svelte application source code
  - `components/` - Reusable Svelte components
  - `routes/` - Route components for the SPA
  - `App.svelte` - Main application component
  - `main.js` - Application entry point
  - `routes.js` - Route configuration

## Dependencies

The web application uses:
- Svelte - Frontend framework
- svelte-spa-router - Single Page Application routing
